## Text preprocessing steps

1. Segmentation.
2. Removal of punctuations or special characters.
3. Lowercasing.
4. Tokenization.
5. Parts-of-Speech (POS) Tagging.
6. Removing Stopwords.
7. Stemming.
8. Lemmatization.

## Segmentation

It is the process where paragraphs are broken into sentences.
